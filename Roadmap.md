# 2026 Roadmap

In late 2025, a vote was held in Pydata groups for our first community project, and a system for missing people and dangerous roads arose in victory. We intend to build a centralized mobile application that would make dissemination of information and locating victims of crimes easier in this current unsettling environment in Nigeria, rather than having scattered information on X, WhatsApp, and Facebook. We hope to launch an MVP before the end of 2026 Q3, that being September 30th. This file provides an overview of the direction this project is heading.

## Q1 to Q2 - Infrastructure

The expected features are the following:

**Account Creation** - Personal information requirement and verification

**Admin Panel** - Post management (deletion, approval, closing of cases, alert range), User Management (suspending/deleting users), Report generation - crime pattern analysis (victim ages, victim genders, hotspot areas, accident-prone roads)

**User Abilities** - Submit post (create requirements for post approval), Receive notifications, DMs, Report/flag content

**Platform/Posts** - Auto threads, Danger warnings for DMs & posts, Case updates, Search functionality, Map/geolocation integration

## Q1 to Q2 - User Interface

Creation of interface for people to easily interact with the features in infrastructure

## Q1 to Q3 - Legal & Compliance

- Terms and conditions
- Lawyers and police discussions
- GDPR/NDPR 2023 privacy law compliance
- Content moderation guidelines
- Launch approval from relevant authorities
